# Dog Heart Size Detection Project 🐶

This project detects the size of a dog's heart (Large, Normal, or Small) from chest X-ray images using deep learning.

We used a model called **ResNet-18** and trained it using PyTorch in Google Colab.  
Our model achieved an accuracy of **76%**, which is better than the 75% baseline in the research paper we referred to.

---

## 📁 Files in this project:

- `Project One_MohammedUvais.ipynb` → Main notebook with code for training, evaluation, and predictions.
- `dog_heart_model.pth` → The trained model saved from Colab.
- `test_predictions.csv` → (Optional) CSV file with predictions for test images.

---

## 🔧 How to Use:

1. Open the notebook in Google Colab.
2. Mount Google Drive and upload your dataset.
3. Run the cells to train the model and evaluate accuracy.
4. Use the model to predict heart size from new X-ray images.
